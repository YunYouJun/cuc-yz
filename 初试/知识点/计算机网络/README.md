### 传输层提供的服务

P204-205 了解传输层寻址与端口（P205 上方 常用端口号和对应应用程序务必牢记，考试可能问端口号对应程序。也可以直接翻到 P266 ，看表 6.2 更全一些）

P205-206 了解无连接服务与面向连接服务（P206 上方 明白 IP 数据包与 UDP 数据报区别）

> TCP：FTP、HTTP、TELNET
>
> UDP：DNS（域名解析）、SNMP

### UDP协议

P208-210 了解 UDP数据报（记忆 P209  上方 UDP 的优点，尤其是相较于 TCP 的部分，格式大致了解即可，考试考的话会给相应的图）

P210 了解UDP校验，可练习课后习题（不常见，但不排除考的可能）

### TCP协议

P215-216 了解 TCP 报文段（格式大致了解，各字段所占大小最好能记住。记忆 P215 上方 TCP 协议的优点，和 UDP 对比着看。）

P219-220 了解 TCP 可靠传输

P220-221 了解 TCP 流量控制

P221-224 掌握拥塞控制（同样会考计算题，不过了解后很容易做。）
